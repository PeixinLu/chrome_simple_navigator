# Chrome Simple Navigator

**语言选择:** [English](./README-EN.md) | [简体中文](./README-ZH-CN.md) | [Bilingual / 双语](./README.md)

欢迎使用 **Chrome Simple Navigator**，这是一款轻量级浏览器扩展，通过**可自定义的键盘快捷键**增强您的浏览体验，实现**跨标签页**级别的 `前进/后退` 导航体验。

## 🎯 功能特性

- **智能后退导航**:按 `Alt + Left`(Windows/Linux)或 `Command + Left`(macOS)在页面历史中后退。如果没有历史记录,将出现确认提示。再次按快捷键可关闭当前标签页。

- **智能前进导航**:按 `Alt + Right`(Windows/Linux)或 `Command + Right`(macOS)在页面历史中前进。如果无法前进,再次按快捷键可恢复最后关闭的标签页。

- **智能标签页管理**:当关闭通过前进导航恢复的标签页时,自动返回到恢复操作前的原始标签页。

- **友好的通知提示**:页面内的弹窗通知(非系统通知)会在需要确认操作时提醒您。

## 🚀 安装

### 手动安装

1. **下载或克隆**此仓库到本地

2. **打开 Chrome** 并访问 `chrome://extensions/`

3. **启用开发者模式**,点击右上角的开关

4. **点击"加载已解压的扩展程序"**按钮

5. **选择**扩展目录(项目根文件夹)

6. 扩展现在应该出现在扩展列表中并处于激活状态

7. 返回页面并刷新

### 权限说明

插件完全离线，不会向任何服务器发送数据，确保隐私安全

扩展需要以下权限:

- `tabs` - 用于导航浏览器历史记录和管理标签页
- `sessions` - 用于恢复最近关闭的标签页
- `activeTab` - 用于与当前活动标签页交互
- `scripting` - 用于注入内容脚本以处理键盘事件
- `storage` - 用于保存扩展首选项

## 📖 使用说明

### 基本导航

- **`Alt + Left` / `Command + Left`**:在页面历史中后退

- **`Alt + Right` / `Command + Right`**:在页面历史中前进

- 可在配置中自定义键盘快捷键

- 可在配置中自定义键盘快捷键

### 高级用法

#### 关闭标签页

1. 当处于页面历史记录的起始位置时,按 `Alt + Left`(Windows/Linux)或 `Command + Left`(macOS)

2. 将出现弹窗通知:"再次按后退以关闭此标签页"

3. 在配置的延迟时间内(默认 1 秒)再次按快捷键以确认并关闭标签页

#### 恢复已关闭的标签页

1. 当没有可用的前进历史记录时,按 `Alt + Right`(Windows/Linux)或 `Command + Right`(macOS)

2. 将出现弹窗通知:"再次按前进以重新打开最后的标签页"

3. 在配置的延迟时间内再次按快捷键以确认并恢复最后关闭的标签页

#### 返回原始标签页

当关闭通过 `Alt + Right` / `Command + Right` 恢复的标签页时,扩展会自动切换回执行恢复操作前的原始标签页。

## ⚙️ 配置选项

访问扩展选项:

1. 右键点击 Chrome 工具栏中的扩展图标

2. 选择"选项"

3. 配置您的首选项:
   - **标签页操作** - 启用/禁用标签页关闭和重新打开功能
   - **弹窗通知** - 启用/禁用确认提示并调整延迟时间
   - **键盘快捷键** - 通过 Chrome 的快捷键设置自定义快捷键
   - **导航日志** - 在弹窗中查看最近的导航操作

## 🔧 开发

### 项目结构

```
chrome_extension/
├── manifest.json          # 扩展清单配置
├── _locales/              # 国际化
│   ├── en/                # 英文翻译
│   │   └── messages.json
│   └── zh_CN/             # 中文翻译
│       └── messages.json
├── src/
│   ├── background/        # 服务工作线程(后台脚本)
│   │   └── index.js
│   ├── content/           # 内容脚本(注入网页)
│   │   └── toast.js
│   ├── popup/             # 扩展弹窗界面
│   │   ├── popup.html
│   │   ├── popup.css
│   │   └── popup.js
│   └── options/           # 选项页面
│       ├── options.html
│       ├── options.css
│       └── options.js
└── assets/
    └── icons/             # 扩展图标
```

### 构建

此扩展使用 Manifest V3,无需构建步骤。只需在 Chrome 中加载未打包的扩展目录即可进行开发。

### 测试

1. 在开发者模式下加载扩展

2. 导航到任何常规网页

3. 测试键盘快捷键并验证导航行为

4. 检查浏览器控制台的调试日志(以 `[background]` 为前缀)

## ⚠️ 限制

- **确认超时**:确认提示会在配置的延迟时间后过期(默认 1 秒)

- **单次会话**:当标签页被移除或浏览器重启时,标签页恢复映射会被清除

- **语言显示**:扩展语言由浏览器的语言设置决定(英文或简体中文)
