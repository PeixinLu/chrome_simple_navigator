# Chrome Extension Starter\n\nThis project bootstraps a Manifest V3 Chrome extension with separate modules for the background service worker, content scripts, popup UI, and options page.\n\n## Structure\n- \\manifest.json\\: Core extension configuration.\n- \\src/background\\: Service worker logic.\n- \\src/content\\: Scripts injected into pages.\n- \\src/popup\\: Browser action popup UI.\n- \\src/options\\: Dedicated options page UI.\n- \\ssets\\: Static assets such as icons.\n\n## Getting Started\n1. Load the folder in Chrome via *chrome://extensions* with Developer Mode enabled.\n2. Choose \\Load unpacked\\ and select this directory.\n3. Modify the modules inside \\src/\\ to build your functionality.\n\n## Development Tips\n- Keep background logic lean and delegate UI to popup/options.\n- Use Chrome Storage APIs for cross-context state.\n- Update \\manifest.json\\ when you add permissions or resources.\n
